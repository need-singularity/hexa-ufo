## §1 4-단계 사다리 — 식당 비유

UFO 같은 건 너무 멀고 어려우니까, **레시피 → 식당 개업** 비유로:

| 단계 | 레시피/요리 비유 | hexa-ufo 비유 | 우리가 한 것 |
|:-----|:-----|:-----|:---:|
| **T1** | 레시피 글로 정확히 적기 (소금 5g, 200℃ 10분) | 종이에 수식 (σ·φ = n·τ = 24) | ✅ |
| **T2** | 머릿속/컴퓨터로 시뮬레이션 ("이렇게 만들면 될 것 같다") | math_pure 로 SI 단위 계산 | ✅ |
| **T3** | 비슷한 요리 만들어 본 다른 사람들 후기 모아 비교 (미슐랭 가이드, 백종원 영상) | Casimir 1948 / Lamoreaux 1997 / Hipparcos 측정값과 일치 확인 | ✅ |
| **T4** | **진짜 식당 차려서 손님한테 팔아보기 — 매일 밤 검증** | **진짜 UFO 만들어서 α-Cen 16일 갔다오기** | ❌ |

T1~T3 다 통과해도 식당 안 차리면 "이론적으로 맛있는 가게" 그치죠. 손님 받기 전엔 진짜 맛있는지 모름.

## §2 hexa-ufo Stage 별 T4 = 무엇을 만들어야 하는가

7단계 추진 스택의 T4 = **각 단계를 실제 하드웨어로 구현**:

| Stage | 우리가 한 것 (T1+T2+T3) | T4 = 실제로 필요한 것 |
|:------|:------|:------|
| **1. 호버 (Meissner)** | RT-SC 48T 가능하다는 수식 | 진짜 상온 초전도체 만들어서 자석 위에 띄우기 |
| **2. 순항 (핵융합)** | 8.7 kW p-¹¹B 가능하다는 계산 | 진짜 탁상용 핵융합 발전기 만들기 |
| **3. 궤도 (반물질)** | 초당 1.7×10¹² 반양성자 가능 | 진짜 반물질 공장 + 감마선 로켓 만들기 |
| **4. 워프 (Alcubierre)** | 마이너스 에너지 σ²=144 만큼 줄일 수 있음 | 진짜 음에너지 만들어서 워프 버블 형성 |
| **5. 웜홀 (Morris-Thorne)** | 목구멍 σ·ℓ_Pl 안정화 가능 | 진짜 안정된 웜홀 통로 뚫기 |
| **6. 차원도약 (KK)** | 4D→26D 사다리 가능 | 진짜 LHC 14TeV 너머에서 KK 입자 검출 |
| **7. 종합 (warp+dim)** | (σ-φ)²=100c, α-Cen 16일 | **진짜 100c 우주선 타고 알파 센타우리 갔다오기** |

## §3 왜 T4 는 안 했나? (물리적으로 인류가 못 함)

- **Stage-1 (상온 초전도체)**: 2023년 LK-99 떠들썩했지만 다 가짜로 판명. 2026년 현재 미해결.
- **Stage-2 (탁상용 핵융합)**: 2023년 NIF 손익분기 잠깐 넘었지만 발전기는 아님.
- **Stage-3 (반물질 공장)**: CERN 이 2024년에 만든 양 = 1초당 ~10¹⁰개. 우리 목표 (1.7×10¹²) 보다 100배 부족.
- **Stage-4~7**: **수학적으로만 정의된 상태** — Alcubierre 1994 논문 이후 30년 넘었지만 진짜 워프 버블 본 사람 없음. 웜홀 본 사람 없음. KK 입자 LHC 검출 못함.

Recipe §9 가 명시: T4 는 **"Mk.V 2150+ SF-class target"** — 즉 2150년 이후나 가능할 일.

## §4 단계별 하드웨어 비교 (현재 vs T4)

### Stage-1 호버 (Meissner 부양)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: 액체질소 (-196℃) 부어야 자석 위에 떠 있는    │
│        조그만 검은 디스크 (YBCO, 1986년 발견)       │
│                                                     │
│  T4:  실온 (25℃) 책상 위에서 그냥 자석 위에 떠 있는 │
│        디스크. 차가 그 위에서 둥둥 떠다님           │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: 저온 초전도체 (MRI 스캐너 안에 들어있음 — 영하 269℃ 액체헬륨)
- **갭**: 상온에서 자기장 48 테슬라 견디는 재료
- **2023 사건**: 한국 LK-99 — 결국 가짜였음 (실제로는 황화구리 불순물)

### Stage-2 순항 (탁상용 핵융합)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: ITER (프랑스, 25년째 짓는 중, 축구장 크기)   │
│        2025년에야 첫 플라즈마, 발전 안 됨           │
│                                                     │
│  T4:  냉장고 옆 핵융합 발전기. 수돗물 같은 중수소만 │
│        넣으면 평생 전기 공짜                        │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: 토카막 (ITER, KSTAR 한국) — 수십 초 플라즈마 유지
- **갭**: 손익분기 (Q>1) 지속 가능 + 작아지기 + 발전기 출력
- **2022 사건**: NIF (미국) — 0.5초간 Q=1.5, 하지만 레이저는 100MJ 들어감

### Stage-3 궤도 (반물질 감마 로켓)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: CERN ALPHA — 연구실 한 칸짜리, 초당 ~100억개 │
│        반양성자 (저장 시간 1000초)                  │
│                                                     │
│  T4:  도쿄돔 크기 반물질 공장 + 1g 반물질 보관통    │
│        (1g 폭발력 = 히로시마 원폭 3배)              │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: CERN antiproton decelerator (1초당 약 10¹⁰개)
- **갭**: 100배 더 빨리 만들기 + 1g 안전 보관 (현재 인류 총 보유량 = 나노그램 단위)
- **사실**: NASA 가 1g 만드는 데 드는 비용 = ~$25조 (2010년 추정)

### Stage-4 워프 (Alcubierre 버블)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: Casimir 판 두 장 사이 (1μm 간격) → 1.3 mPa  │
│        흡인력 (Lamoreaux 1997, 측정 가능)          │
│                                                     │
│  T4:  반경 1m 버블 만들 음에너지 ≈ M_galaxy 분량   │
│        VDB σ²=144 줄여도 ≈ 태양 질량                │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: Casimir 효과 측정 (1948 예측, 1997 검증)
- **갭**: M_☉ 규모 음에너지를 ℓ_Pl/12 두께 벽에 농축 — Pfenning-Ford QI bound 위반
- **현실**: 어떤 양자장 source 도 아직 macroscopic NEC violation 못함 (F-WARP-1 OPEN)

### Stage-5 웜홀 (Morris-Thorne 통로)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: 양자 거품 (Planck 스케일에 가상으로 잠깐    │
│        생겼다 사라지는 것 — 측정 불가)              │
│                                                     │
│  T4:  사람이 걸어 들어갈 수 있는 안정 웜홀 (b₀≥1m) │
│        + Hawking chronology protection 우회         │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: 수학적으로 정의됨 (Morris-Thorne 1988)
- **갭**: exotic matter source + back-reaction 안정화 + CTC 방지
- **현실**: 지금까지 자연에서 웜홀 한 번도 관찰 안 됨 (F-WORM-1/2/3 OPEN)

### Stage-6 차원도약 (KK 입자)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: LHC (둘레 27km, 14 TeV) — KK 입자 못 봄     │
│        Eöt-Wash 토션밸런스 — 50μm 까지 표준 중력   │
│                                                     │
│  T4:  FCC (둘레 100km, 100 TeV, 2070+ 계획) 또는   │
│        sub-mm 중력 정밀 측정 (10⁻⁹ m 까지)         │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: LHC 데이터 (KK 후보 없음, m_KK > 14 TeV 하한)
- **갭**: 더 높은 에너지 또는 더 정밀한 sub-mm 중력 측정
- **현실**: 우리 R_c = 1728·ℓ_Pl ≈ 2.8×10⁻³² m — Eöt-Wash 감도보다 10²⁶배 작음

### Stage-7 통합 (warp + dimjump 합성)

```
┌─ 사진 비유 ────────────────────────────────────────┐
│  현재: 보이저 1호 (1977 발사, 49년째 비행 중,      │
│        17 km/s ≈ 0.0000567c)                        │
│                                                     │
│  T4:  100c 우주선 → α-Cen 16일, Sirius 31일        │
│        Mk.V 2150년+ "SF-class target"               │
└────────────────────────────────────────────────────┘
```

- **이미 있음**: Voyager (느림) / SpaceX Starship (planet-bound)
- **갭**: warp + dimjump 둘 다 작동 + 합성 안정성
- **현실**: UNPROVEN² — 두 upstream 모두 unproven (F-USE-{1..4} OPEN)

## §9 정리

```
물리 사양 ─→ 수치 사양 ─→ 논문 anchor ─→ ❌ 설계도 ─→ ❌ 시제품 ─→ ❌ 양산
   T1         T2          T3            ⤺ 여기 갭        T4 hardware
   ✅ 100%     ✅ 100%      ✅ 100%
                                     ⬆
                            hexa-ufo 가 도달한 한계점
```

**hexa-ufo 는 "수학적 청사진" 까지만.** 진짜 설계도 (CAD + BOM + 시공
절차서) 만들려면:

1. **Stage-1**: 진짜 RT-SC 재료 발견되어야 (그 다음 자석 코일 설계 가능)
2. **Stage-2**: 핵융합 노형 결정 (토카막? 관성? IEC?) 후 ITER 도면 참고
3. **Stage-3**: 반물질 가속기 + 트랩 도면 (CERN ALPHA 확장)
4. **Stage-4~7**: **수학적으로도 아직 모름** — 음에너지 source 발견되기 전엔 설계도 자체가 불가능

> 즉 hexa-ufo 는 **"무엇을 만들어야 하는지의 수치 사양"** 까지만 적었고,
> **"어떻게 그것을 만드는지의 도면"** 은 인류 전체가 아직 그릴 수 없는
> 단계.
